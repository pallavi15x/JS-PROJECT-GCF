Q: Difference between Struct and Union in C?

Posted by: Rohan Roy | 3 hours ago

I am confused about how memory is allocated for unions compared to structs when dealing with different data types.

Faculty Answer (Prof. S. Sen): Struct allocates distinct memory locations for all members, whereas Union shares a single common memory space size equal to its largest member.

Q: Pure HTML/CSS responsive grids without JavaScript?

Posted by: Kabir Singh | 2 days ago

Is it possible to make a multi-page portfolio fully responsive using only semantic HTML and CSS? I want to avoid JS for layout shifts.

Peer Answer (Alex M.): Absolutely! Use CSS Grid with `grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));`. It automatically handles the wrapping for mobile screens without a single line of JavaScript. Also, look into CSS media queries (`@media`) for fine-tuning margins!

Q: Norton's vs Thevenin's Theorem for complex circuits?

Posted by: Neha Verma | 4 days ago

When analyzing a circuit with multiple voltage and current sources, is it better to convert everything to a Thevenin equivalent or a Norton equivalent?

Faculty Answer (Dr. A. Gupta): They are mathematically interchangeable via source transformation (V_th = I_n * R_th). However, if your load is in parallel with the rest of the circuit, Norton's (current source) often makes the final nodal analysis easier.
